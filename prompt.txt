PROMPT 1
================================================================
Problem Statement: Write a 2D graphics editor using * and _.

Write functions to draw a circle, rectangle, line, and triangle.
You should be able to:
  - Add objects to the picture
  - Delete objects from the picture
  - Modify objects in the picture

Use a 2D array of characters to store the picture.
Write a function to display the picture.

----------------------------------------------------------------
ANSWER 1
----------------------------------------------------------------
An implementation plan was created for the 2D graphics editor with
the following proposed design:

  Proposed Architecture:
  - A 2D character canvas (e.g., 40 rows x 80 cols) filled with spaces.
  - A Shape data structure (union/struct) to store shape type and params.
  - A global shapes[] array (up to MAX_SHAPES) to track all added objects.
  - Functions:
      * draw_line(x1, y1, x2, y2)       - Bresenham's Line Algorithm
      * draw_circle(cx, cy, radius)     - Midpoint Circle Algorithm
      * draw_rectangle(x, y, w, h)      - Border drawing with * and _
      * draw_triangle(x1,y1,x2,y2,x3,y3) - Three lines connecting vertices
      * display_canvas()                - Print the 2D array row by row
  - Menu system:
      1. Add Shape
      2. Delete Shape
      3. Modify Shape
      4. Display Canvas
      5. Exit

ANSWER 2
----------------------------------------------------------------
The implementation plan was approved and the full C code was written
to main.c. Here is a summary of what was implemented:

  Summary of Work Done:
  - Dynamic Shape Manager:
      Added a static array Shape shapes[MAX_SHAPES] that tracks all
      shapes. The canvas redraws dynamically when the list changes.

  - Rasterization Algorithms:
      * Bresenham's Line Algorithm  -> draws lines using * characters
      * Midpoint Circle Algorithm   -> draws circles using * characters
      * Rectangle border drawing    -> uses * for corners/sides and _
                                       for top/bottom edges
      * Triangle drawing            -> connects three vertices using
                                       the line drawing function

  - Menu and Input Validation:
      Added an interactive terminal menu with strict input validation
      to prevent scanf from getting stuck in infinite loops.

  - Files Created:
      * task.md      - Tracks implementation progress
      * walkthrough.md - Documents how to compile and run the program

  The full source code is available in main.c.

ANSWER 3
----------------------------------------------------------------
There is no host C compiler (such as gcc, clang, or MSVC's cl.exe)
installed or configured in the system's PATH or standard folders.
As a result, the C source code cannot be compiled into a runnable
executable directly from this environment.

To run the updated 2D graphics editor, the code must be compiled
manually. Here is how:

  1. Install a C Compiler:
     - Windows (MinGW-w64):
         Download and install a lightweight compiler toolchain like
         w64devkit from: https://github.com/skeeto/w64devkit/releases
         or use MSYS2 from: https://www.msys2.org/
     - Visual Studio:
         If Visual Studio is installed, use the "Developer Command
         Prompt for VS" which has cl.exe configured.

  2. Compile the code:
     Open a terminal in the project directory and run:

         gcc main.c -o editor.exe

     Or if using MSVC:

         cl main.c /Fe:editor.exe

  3. Execute the compiled program:

         .\editor.exe

the automated install.

  Resolution / What to do next:

  Option A - Download w64devkit directly:

    gcc main.c -o editor.exe
